# SPNL Bisection - Streamlit App

Aplikasi web sederhana untuk mencari akar fungsi menggunakan Metode Bisection.

## Cara menjalankan (lokal)
1. `pip install -r requirements.txt`
2. `streamlit run app.py`

## Deploy
Deploying to Streamlit Cloud: create repo on GitHub and connect via https://share.streamlit.io
